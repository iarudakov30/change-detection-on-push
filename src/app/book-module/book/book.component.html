To work successfully with OnPush CD without problems follow this steps: 1. Keep business logic in "container" and rendered logic in "component" 2. Keep in service huge data calculating and subscriptions 3. Work in dynamic programming flow, avoid ".subscribe()" or use it only when needed 4. Get and keep data in container as Observable streams 5. Render data in template or throw it to child component via "async" pipe 6. Take data from parent component via @Input() 7. Add 'changeDetection: ChangeDetectionStrategy.OnPush' to @Component() meta data 8. Use *ngIf in forms/tables while you data is loading 9. Do not mutate the data directly, use BehaviorSubject or similar
